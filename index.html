Description

This Amazon Book Review Analyzer uses machine learning to automatically classify book reviews into three sentiment categories: Bad Review, Mixed Review, and Great Review.

Model Information

Property Details
Model Type Logistic Regression
Accuracy 84.5% on test dataset
Features TF-IDF Vectorization with 5000 features
Preprocessing Lowercasing, special character removal, stopword removal, lemmatization
Classes üî¥ Bad Review (Negative): 1-2 stars
üü° Mixed Review (Neutral): 3 stars
üü¢ Great Review (Positive): 4-5 stars

Dataset: Books_ratings.csv

Total Reviews: Approximately 29,352 book reviews from Amazon

Original Features:

  • Id: Unique identifier for the book
  • Title: The title of the book
  • Price: Book price (some missing values)
  • User_id: Unique identifier for the reviewer
  • profileName: Display name of the user
  • review/helpfulness: Helpfulness rating information
  • review/score: Numerical rating (1-5 stars)
  • review/time: Review timestamp
  • review/summary: Short review summary
  • review/text: Full review content

Derived Features:

  • sentiment: Categorical label ('Negative', 'Neutral', 'Positive') derived from review/score:
    • 1-2 stars ‚Üí Negative
    • 3 stars ‚Üí Neutral
    • 4-5 stars ‚Üí Positive
  • cleaned_review: Processed text after lowercasing, removing special characters, stopword removal, and lemmatization

Primary Use: Sentiment analysis based on review/text and review/score columns using TF-IDF vectorization and Logistic Regression model.
